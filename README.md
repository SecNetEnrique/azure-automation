# Azure Automation Lab

This repository contains hands-on cloud automation projects using Microsoft Azure, PowerShell, and Infrastructure-as-Code tools.

## Purpose
To demonstrate practical skills in:
- Azure administration
- Cloud security fundamentals
- Automation using PowerShell
- Infrastructure-as-Code (Bicep/Terraform)
- Identity and access management (Entra ID / Intune)

## Structure

- `powershell/` → Automation scripts for Azure, Intune, and Entra ID
- `bicep/` → Infrastructure-as-Code templates
- `terraform/` → Cloud infrastructure deployments
- `scripts/` → Utility scripts (monitoring, cleanup, backups)
- `docs/` → Architecture diagrams and documentation

## Focus Areas

- Virtual Machine deployment automation
- Identity and access management
- Security policy enforcement
- Cloud monitoring and operations

## Status
Active learning and portfolio development project.
